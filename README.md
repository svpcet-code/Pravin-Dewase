# Pravin Dewase's Birthday Celebration ✨

A premium cinematic birthday memory website built with Next.js, Tailwind CSS, Framer Motion, GSAP, Lenis, and Three.js.

## Features

- **Cinematic opening intro** — GSAP curtain reveal, light streaks, particle effects
- **Hero section** — Parallax café background, mouse-reactive glow, animated CTA
- **Premium gallery** — Masonry layout with blur-to-clear loading and hover zoom
- **Family cards** — 3D tilt on mouse movement with glassmorphism
- **Animated timeline** — Scroll-triggered GSAP animations with connecting line
- **Emotional ending** — Cinematic fade with final thank-you message
- **Ambient effects** — Three.js golden particles, stars, custom cursor, scroll progress

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js 15 (App Router)
- Tailwind CSS 4
- Framer Motion
- GSAP + ScrollTrigger
- Lenis smooth scrolling
- Three.js / React Three Fiber
- React Icons

## Customization

Replace gallery images in `src/components/sections/GallerySection.tsx` with your own family photos. Update the `remotePatterns` in `next.config.ts` if using a different image host.
